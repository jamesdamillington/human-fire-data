# Human Fire Data

This repository contains code for analysing three global databases of human fire, plus data from the Global Fire Emissions Database:
- GFED.r 
- mapping.r 

It also contains high-resolution figures from outputs of code that were subsequently edited in inkscape, as follows:

[Figure: DAFI+GFUS](png/Fig-DAFI-GFUS.png). Suppression Level from DAFI (case study points) and GFUS (region polygons) data.

[Figure: GFED+LIFE](png/Fig-GFED-LIFE.png). Difference in burned area (cell fraction) between GFED4s and GFED5 products (grid data) with location of LIFE case study locations (points).
